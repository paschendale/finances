# AI_GUIDELINES.md

This repository is designed to be developed with multiple AI coding agents.

Before implementing any feature, agents must read:

RULES.md

and the documentation in:

docs/

Key documents:

docs/PRODUCT.md
docs/DOMAINS.md
docs/LEDGER_RULES.md
docs/ARCHITECTURE.md

Agents must follow the workflow defined in RULES.md.

Required workflow:

1. Understand the product requirements.
2. Implement the feature.
3. Record the implementation in IMPLEMENTATION_LOG.md.
4. Record unresolved issues in LEDGER.md.

Do not introduce new frameworks or architecture changes unless explicitly required.

Prefer simple implementations aligned with the existing architecture.
